# 슬라이드 1 / 6 — 프로젝트 개요

## 타이틀

**ReviewTime**
AI 기반 리뷰 분석 기능

> 리뷰 100개를 읽는 데 걸리는 시간? — **3초**

---

## 문제 정의 → 솔루션

| 문제 | | 솔루션 |
|---|:---:|---|
| 배달 앱에 리뷰가 쌓인다 | → | AI 주차별 리뷰 분석 |
| 별점만 보고 판단 | → | 주별 트렌드 시각화 |
| 리뷰 텍스트는 비정형 | → | 형태소 분석 (kiwipiepy) |

---

## 기술 스택

- **백엔드**: Django (Python)
- **형태소 분석**: kiwipiepy
- **LLM**: Claude API (최소 토큰 사용)
- **프론트엔드**: Bootstrap 5 + Vanilla JS
- **데이터베이스**: SQLite

---

## 핵심 지표

| 지표 | 수치 |
|---|---|
| 분석 가능 리뷰 수 | 100건/가게 |
| LLM 토큰 절약 | 25배 |
| 로컬 처리 비중 | 75~85% |
| 분석 주기 | 월별 + 주별 (ISO 8601) |
| 카테고리 분류 | 음식/배달/서비스/가격/위생 |

💡 LLM에 전달하는 압축 JSON 예시

// 원문 100개가 아니라 압축된 통계 JSON만 전달
{
  "store": "파스타 팩토리",
  "period": "2026-W13 ~ 2026-W16",
  "avg_rating": 4.0,
  "sentiment": { "positive": 65, "neutral": 25, "negative": 10 },
  "top_keywords": ["리뉴얼", "알덴테", "크림소스", "바삭한"],
  "trend": "up",
  "rating_change": "+2.0 (W13 대비 W16)"
}

※ 실제 AI 요약: "파스타 팩토리의 2026-W13 ~ 2026-W16 리뷰를 종합 분석한 결과, 평균 평점 4.0점(긍정 65%, 부정 10%)을 기록했습니다. 기간 내 평점이 3.0점에서 5.0점으로 큰 폭 상승하며 빠른 개선세를 보이고 있습니다."

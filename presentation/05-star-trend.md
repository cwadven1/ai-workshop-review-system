# 슬라이드 5 / 6 — 별점 트렌드 시각화

## 타이틀

**별점을 주식 차트처럼**

시가/종가/고가/저가로 표현하는 리뷰 트렌드

---

## 주식형 시각화 개념

| 주식 용어 | 리뷰 적용 | 의미 |
|---|---|---|
| 시가 | 기간 첫 주 평균 별점 | 기간 시작 시점 평가 |
| 종가 | 최근 주 평균 별점 | 현재 평가 |
| 고가 | 기간 내 최고 주 평균 | 최고 컨디션 |
| 저가 | 기간 내 최저 주 평균 | 최저 컨디션 |
| 거래량 | 리뷰 수 | 신뢰도 지표 |
| 이동평균선 | 4주 이동평균 | 단기 노이즈 제거한 추세 |
| 변동률 | 전주 대비 별점 변화% | 개선/악화 속도 |

## 차트 구성 요소

### 메인 라인 차트 (듀얼 축)
```
별점 (좌축)      리뷰 수 (우축)
5.0 ┤    ●                   ┤ 5
4.5 ┤   ╱ ╲  ●               ┤ 4
4.0 ┤  ╱   ╲╱  ←이동평균선   ┤ 3
3.5 ┤ ╱                       ┤ 2
3.0 ●                          ┤ 1
    └─────────────────────────
     W13  W14  W15  W16
```

### 주별 카드
각 주차마다 카드 형태로 표시:
- 평균 별점 (크게)
- 전주 대비 변화 (+/- 숫자)
- 리뷰 수
- 신뢰도 등급 (높음/보통/낮음)

### AI 총평 카드 (최신 주차 상단)
- W16 → W13 역순으로 표시
- 통계 수치 없이 "왜" 변했는지만 설명
- 주요 키워드 버튼 (클릭 시 리뷰 필터)

## 신뢰도 표현 로직

| 리뷰 수 | 신뢰도 | 색상 |
|---|---|---|
| 평균 이상 | 높음 | 초록 |
| 평균의 70~100% | 보통 | 파랑 |
| 평균의 70% 미만 | 낮음 | 주황 (경고) |

> 리뷰 수가 적은 주차의 평점 과신 방지

## 인터랙션 기능

| 기능 | 설명 |

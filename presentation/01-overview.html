리뷰 텍스트는 비정형

음식/배달/서비스 중 어디가 문제인지 텍스트만으로는 빠르게 파악할 수 없다.

솔루션 — 3가지 핵심 기술

🌿

형태소 분석 (kiwipiepy)

로컬 실행으로 API 비용 없이 키워드 추출과 감성 태깅을 자동화합니다.

비용 $0
🤖

LLM 하이브리드 전략

애매한 케이스 15~25%만 LLM에 전달하여 토큰 사용량을 25배 절약합니다.

5,000→200 토큰
📈

주별 트렌드 시각화

별점을 주식 차트처럼 시가/종가/고가/저가로 표현하고 급변 알림으로 위기를 조기 감지합니다.

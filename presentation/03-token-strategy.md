## 역할 분담

### kiwipiepy (로컬, 비용 $0)

- 형태소 분석 (명사/형용사 추출)
- 키워드 빈도 집계
- 간단한 감성 판단 (룰 기반)
- 애매한 케이스 필터링
- 별점 트렌드 계산
- 비교 분석

### LLM (API, 최소 토큰)

- 애매한 리뷰 감성 판단 (이중부정, 혼합감성)
- 키워드 통계 → 자연어 리포트 생성
- W13~W16 기간 종합 AI 요약

---

## LLM에 전달하는 데이터 예시

```json
{
  "store": "파스타 팩토리",
  "period": "2026-W13 ~ 2026-W16",
  "avg_rating": 4.0,
  "sentiment": { "positive": 65, "neutral": 25, "negative": 10 },
  "top_keywords": ["리뉴얼", "알덴테", "크림소스", "바삭한"],
  "trend": "up",
  "rating_change": "+2.0 (W13 대비 W16)"
}
```

> 원문 100개 전체가 아니라 **압축된 통계 JSON** 만 전달

---

## 실제 적용: 기간 종합 AI 요약

views.py의 `_build_recent_analysis()` 함수가 4주 데이터를 분석하여 자동 생성:

- W13~W16 가중 평균 평점 계산
- 긍정/부정 비율 집계
- 상위 키워드 Counter로 집계
- 트렌드 방향 자동 판단
- **단일 종합 요약 문장** 자동 생성

```
"파스타 팩토리의 2026-W13 ~ 2026-W16 리뷰를 종합 분석한 결과,
 평균 평점 4.0점(긍정 65%, 부정 10%)을 기록했습니다.
 기간 내 평점이 3.0점에서 5.0점으로 큰 폭 상승하며 빠른 개선세를 보이고 있습니다.
 자주 언급된 키워드는 '리뉴얼, 알덴테, 크림소스'입니다."
```
